# Tele Coach - Installation and User Guide

## What is Tele Coach?

Tele Coach is a real-time call coaching assistant that detects objections and provides intelligent response guidance during sales calls.

## Installation

### macOS

1. **Download**: Open the `.dmg` file you received
2. **Install**: Drag Tele Coach to your Applications folder
3. **Grant Permissions**: On first launch, allow microphone access when prompted
4. **Launch**: Open Tele Coach from Applications or Launchpad

### Windows

1. **Download**: Run the `.exe` installer you received
2. **Install**: Follow the installer prompts (default location recommended)
3. **Permissions**: Allow microphone access when prompted by Windows Security
4. **Launch**: Tele Coach from Start Menu or desktop shortcut

## First Time Setup

The app is ready to use after installation. The speech engine and coaching content are included; no separate setup or download is required. You may only need to grant microphone access when you first start a coaching session.

### Grant Microphone Permissions

**macOS**:
- Go to System Settings → Privacy & Security → Microphone
- Enable Tele Coach in the list
- Restart the app if needed

**Windows**:
- Go to Settings → Privacy → Microphone
- Allow apps to access your microphone
- Enable Tele Coach in the list

### Finding the Overlay

The Tele Coach overlay appears as a floating window:
- **macOS**: Look for "Tele Coach Overlay" in your menu bar
- **Windows**: Check system tray for Tele Coach icon
- **Position**: Drag the overlay to reposition it

## Using Tele Coach

### Starting a Coaching Session

1. **Start Listening**: Click "Start coaching" in the overlay
2. **Speak Naturally**: The app will transcribe your speech in real-time
3. **Get Coaching**: Objections are detected automatically with suggested responses
4. **Copy Responses**: Click "Copy" to copy text to clipboard

### Overlay Modes

- **Compact**: Shows essential coaching information
- **Expanded**: Includes full transcription history
- **Toggle**: Click "Toggle overlay mode" to switch between views

### Copying Responses

- Click any "Copy" button to copy that text to your clipboard
- Paste into your CRM, email, or notes

### Testing without microphone

Open **Settings** (from the tray or overlay) and use the **Manual test** field: enter a phrase such as "We already use Experian" or "Not interested at all" and run the test. The overlay will update with the detected objection and suggested response, question, and bridge, so you can verify coaching without speaking.

## Platform Notes

### macOS

- **Apple Silicon**: Works natively on M1/M2/M3 Macs
- **Intel Mac**: Full compatibility with Intel-based Macs
- **Permissions**: May need to grant microphone access in System Settings

### Windows

- **Windows 10/11**: Full compatibility
- **Antivirus**: May need to allow Tele Coach through security software
- **Microphone**: Check Windows Privacy Settings if microphone isn't detected

## Getting Help

### Common Issues

- **Overlay not visible**: Check if Tele Coach is running (menu bar/system tray)
- **No transcription**: Ensure microphone permissions are granted
- **Poor accuracy**: Speak clearly and reduce background noise

### Support

For technical support or questions, contact your system administrator.
